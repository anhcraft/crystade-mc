# Crystade MC

This monorepo hosts Minecraft plugins/mods that integrate [Crystade](https://crystade.com) -  a SaaS platform for cron jobs, monitoring and health checks.

![Banner](./assets/banner.jpeg)

## Getting Started

Read the guide: [crystade/push-config-example](https://github.com/crystade/push-config-example)

## Features
- Push configuration: Configure your Crystade tenant from the server's config folder
- Passive monitoring (coming soon)
- Self-hosted agent (coming soon)

## Commands

- `/crystade push`: push the configuration (requiring `crystade.push`)

## Configuration

```yaml
api-key: "your_api_key"
```
